# List-Of-Contacts

### **Overview**
A simple fullStack app that show 10 users that logged on. 

## Installation

1. clone git@github.com:Saeed-Awessat/List-Of-Contacts.git
2. install npm via "npm install"
3. install MongoDb on your computer
3. go to client folder and install npm via "npm install"

## Run App 
1. run mongodb locally
2. start from the root folder "List-Of-Contacts":
    a. run the server via "node server/server"
    b. open new terminal and run the client via "npm start"
